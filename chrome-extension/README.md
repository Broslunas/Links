# Broslunas Links - Chrome Extension

This extension allows you to quickly shorten URLs using your Broslunas Links instance.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select the `chrome-extension` folder in your project root.

## Setup

1. Click the extension icon in your browser toolbar.
2. Go to Settings (gear icon).
3. Enter your API Token (can be generated from your Broslunas Links dashboard).
4. Set the API Endpoint (default is `http://localhost:3000`).
5. Save settings.

## Features

- **One-click shortening**: Pulls the current tab's URL automatically.
- **Custom Slugs**: Set a custom alias for your shortened link.
- **Auto-copy**: Shortened URLs are automatically copied to your clipboard.
- **Context Menu**: Right-click any link to shorten it (background task).
- **Premium Design**: Dark/Light mode support with modern aesthetics.
